Sajad Kebab Persia

از سال ۱۳۵۷ – از شیراز تا جاکارتا

Sejak 1978, api kami tak pernah padam.

Shiraz, 1955

Di gang kecil pasar Vakil, seorang anak 8 tahun bernama Hossein Sajad belajar dari kakeknya: “Kebab bukan daging yang dibakar. Kebab adalah janji antara api dan hati yang lapar.”

Jakarta, 1978

Dengan dua koper dan tungku arang kecil, Hossein mendarat di Jakarta. Di Glodok ia menyalakan api pertamanya. Malam itu hanya 3 orang yang membeli. Seminggu kemudian, antrean sudah mengular sampai ke ujung jalan.

Api yang Tak Pernah Padam Selama 47 Tahun

Bara yang dinyalakan tahun 1978 masih hidup sampai hari ini.
Ditambah sedikit arang baru setiap hari — tapi intinya sama.
Sama seperti cinta Jakarta kepada kami.

Api Kami Tak Pernah Padam

Supaya kapan pun kamu datang — capek, patah hati, atau laper jam 2 pagi —
ada rumah yang menunggumu.
